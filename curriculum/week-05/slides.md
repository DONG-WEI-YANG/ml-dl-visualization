## Slide 2: 從回歸到分類
### 線性回歸不適合分類的三個原因
| 問題 | 影響 |
|------|------|
| 輸出範圍無限制 | 無法解釋為機率 |
| 對離群值敏感 | 決策邊界被極端值拉偏 |
| 缺乏機率語義 | 無法回答「有多大把握？」 |

**解法：** 用 Sigmoid 函數將輸出壓縮到 (0, 1)

---

## Slide 3: Sigmoid 函數
### $\sigma(z) = \frac{1}{1 + e^{-z}}$
```
     1 ─────────────────── ·····
                        ····
                     ···
                  ··
     0.5 ───── ·· ──────────────
             ··
          ···
       ····
     0 ·····────────────────────
        -6  -4  -2  0  2  4  6
```
- 值域 (0, 1)：天然的機率解釋
- 導數：$\sigma'(z) = \sigma(z)(1 - \sigma(z))$
- 中心對稱：$\sigma(-z) = 1 - \sigma(z)$

---

## Slide 4: 邏輯迴歸模型
### Logistic Regression = Linear + Sigmoid
$$P(y=1|x) = \sigma(w^T x + b)$$

**訓練目標：** 最小化二元交叉熵損失 (BCE Loss)
$$\mathcal{L} = -\frac{1}{N}\sum_{i=1}^{N}[y_i \ln \hat{p}_i + (1-y_i)\ln(1-\hat{p}_i)]$$

| 真實 $y$ | 預測 $\hat{p}$ | 損失 |
|:-:|:-:|:-:|
| 1 | 0.99 | 0.01 (低) |
| 1 | 0.01 | 4.6 (高!) |

**梯度：** $\nabla_w \mathcal{L} = \frac{1}{N}\sum(\hat{p}_i - y_i)x_i$ — 形式簡潔！

---

## Slide 5: 決策邊界
### Decision Boundary: $w^Tx + b = 0$
```
  x2
   |  x  x     |  o  o
   |    x   x  |    o  o
   |  x   x    |  o    o
   |     x     |    o
   +───────────┼──────────> x1
            邊界線
```
- 閾值 $\tau = 0.5$ 時，邊界就是 $w^Tx + b = 0$
- 2D 中是直線，高維中是超平面
- 本質上是**線性**分類器

**C 值（正則化）的影響：**
| C 小 | C 中 | C 大 |
|:----:|:----:|:----:|
| 簡單平滑 | 適度擬合 | 可能過擬合 |

---

## Slide 6: 閾值選擇的藝術
### 調整閾值 $\tau$ 改變 Precision-Recall 權衡
| 場景 | 策略 | 原因 |
|------|------|------|
| 疾病篩檢 | 降低 $\tau$ | 不想漏掉病患 (提高 Recall) |
| 垃圾郵件 | 提高 $\tau$ | 不想誤判正常信 (提高 Precision) |
| 一般分類 | $\tau = 0.5$ | 平衡兩者 |

> 閾值不是模型的一部分，而是**部署決策**的一部分

## Slide 7: 混淆矩陣 Confusion Matrix
### 分類評估的基石

|  | 預測 Positive | 預測 Negative |
|:---:|:---:|:---:|
| **實際 Positive** | TP (真正例) | FN (偽反例) |
| **實際 Negative** | FP (偽正例) | TN (真反例) |

**四大指標：**
- **Accuracy** = (TP+TN) / 全部 — 整體正確率
- **Precision** = TP / (TP+FP) — 預測為正的有多準？
- **Recall** = TP / (TP+FN) — 正樣本抓到多少？
- **F1** = 2 * P * R / (P + R) — 調和平均

## Slide 8: ROC 曲線與 AUC
### Receiver Operating Characteristic
```
  TPR
   1 ┌───────────────────┐
     │      ╭──────────  │
     │    ╱              │
     │  ╱   ROC Curve    │
     │╱                  │
   0 └───────────────────┘
     0        FPR        1
```
- **橫軸 FPR** = FP / (FP+TN)
- **縱軸 TPR** = TP / (TP+FN) = Recall
- **AUC = 曲線下面積**

| AUC | 意義 |
|:---:|------|
| 1.0 | 完美分類 |
| 0.9+ | 極優 Excellent |
| 0.8-0.9 | 良好 Good |
| 0.5 | 隨機猜測 |

## Slide 9: PR 曲線 — 不平衡資料的首選
### Precision-Recall Curve
```
  Precision
   1 ┌───────────────────┐
     │\                  │
     │ ╲                 │
     │  ╲  PR Curve      │
     │   ╲───╲           │
     │        ╲──        │
   0 └───────────────────┘
     0       Recall      1
```
**為什麼不平衡資料要用 PR 曲線？**
- ROC 中 FPR = FP/大量負樣本 → 分母大，FPR 看起來很小
- PR 中 Precision = TP/(TP+FP) → 直接暴露 FP 的問題
- **經驗法則：** 正類比例 < 20% → 優先看 PR 曲線

---

## Slide 10: 不平衡資料集處理策略
### 三個層面的解法

**1. 資料層面 Data-Level**
| 方法 | 做法 | 注意 |
|------|------|------|
| 過採樣 Oversampling | 增加少數類 | 易過擬合 |
| SMOTE | 合成新少數類樣本 | 最常用 |
| 欠採樣 Undersampling | 減少多數類 | 可能丟失資訊 |

**2. 演算法層面：** `class_weight='balanced'`
**3. 評估層面：** 用 F1/MCC/PR-AUC 取代 Accuracy

---

## Slide 11: 多類別分類擴展
### 從二元到多類別

| 方法 | 分類器數 | 特點 |
|------|:--------:|------|
| **OvR** (One-vs-Rest) | $K$ | 每個類別 vs 其餘，sklearn 預設 |
| **OvO** (One-vs-One) | $K(K-1)/2$ | 兩兩配對，投票決定 |
| **Softmax** | 1 | 直接輸出 K 類機率，神經網路首選 |

**Softmax 函數：**
$$P(y=k|x) = \frac{e^{z_k}}{\sum_{j=1}^{K} e^{z_j}}$$
- 所有機率和為 1
- 當 K=2 時退化為 Sigmoid

## Slide 12: 本週重點回顧
### Key Takeaways
1. **邏輯迴歸** = 線性模型 + Sigmoid → 機率輸出
2. **BCE Loss** 對高信心錯誤施加大懲罰
3. **決策邊界**是線性的，受正則化 C 值影響
4. **閾值選擇**取決於業務需求，不是固定 0.5
5. **混淆矩陣**是所有指標的基礎
6. **ROC/AUC** 衡量整體排序能力，**PR 曲線**在不平衡資料更可靠
7. **不平衡資料**需要特殊處理策略
