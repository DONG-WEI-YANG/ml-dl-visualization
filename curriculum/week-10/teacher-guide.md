## 教學重點 Key Points

1. **超參數 vs 模型參數的類比是關鍵**：用「考試規則 vs 學生答案」的類比讓學生直覺理解差異。如果學生仍有疑問，可以用更多例子：廚師做菜時，食譜中的「火候大小」和「烹飪時間」是超參數，菜的最終味道是模型參數在優化的結果。

2. **Grid vs Random 的視覺化比較非常重要**：Bergstra & Bengio (2012) 的那張經典示意圖（Slide 8）是本週的核心概念之一。確保學生理解「當只有少數超參數重要時，Random Search 在重要維度上探索更多」的洞見。

3. **學習曲線是診斷工具，不是目標**：強調學習曲線的目的是幫助你做決定（要不要收集更多資料？要不要換更複雜的模型？），而不是追求某種「漂亮」的曲線形狀。

4. **驗證曲線幫助找到「甜蜜點」**：用 Slide 13 的圖讓學生建立直覺——大部分超參數都有一個「太小不好、太大也不好」的最佳區間。

5. **計算資源意識**：讓學生體會到超參數調校可以非常耗費計算資源。在實作中使用 `%%time` 計時，讓他們親自感受 Grid Search 和 Random Search 的時間差異。

## 教學策略 Teaching Strategies

### 開場活動（5 分鐘）
請學生思考：「假設你有一個隨機森林模型，目前的準確率是 85%。你有三個選擇：(1) 收集更多資料、(2) 做更好的特徵工程、(3) 調校超參數。你會先做哪個？為什麼？」

- 沒有標準答案，重點是引導學生思考不同策略的成本效益
- 引出學習曲線作為診斷工具的概念

### 理論講解注意事項
- **Grid Search**：在白板上畫 2D 網格，讓學生數總共有多少種組合。然後加第 3 個維度，讓他們感受指數增長
- **Random Search**：在同一個 2D 網格上隨機撒點，讓學生對比兩種方法在重要維度上的覆蓋率
- **Bayesian Optimization**：只需要講概念，不需要推導數學。用「假設你在一個房間裡找最暗的角落，但你只能開有限次手電筒」的比喻
- **學習曲線**：準備三張事先畫好的學習曲線圖（欠擬合、過擬合、良好擬合），讓學生先猜各自代表什麼狀況

### 實作引導
1. **先示範 GridSearchCV 的基本用法**（5 分鐘），然後讓學生自己嘗試
2. **學習曲線繪製是重點**：確保每位學生都能畫出並正確解讀
3. **熱力圖是本週的視覺化亮點**：展示 2D 超參數搜尋結果的熱力圖，讓學生看到「效能景觀 (Performance Landscape)」
4. **計時比較**：讓學生同時跑 Grid Search 和 Random Search，比較時間

## AI 助教設定 AI Tutor Configuration
本週助教設定為「引導模式」：
- 當學生詢問「最佳超參數是什麼」時，引導他們思考搜尋策略而非直接給值
- 強調不同資料集/模型的最佳超參數不同，沒有放諸四海皆準的答案
- 提示學生使用學習曲線和驗證曲線作為診斷工具
- 當學生的搜尋空間設計不佳時（如使用線性尺度搜尋學習率），給予提示
- 鼓勵學生先分析問題再決定搜尋策略

## 常見問題與排除 Troubleshooting

### Q1: GridSearchCV 執行時間太長
- 減少搜尋空間（先示範小規模的例子）
- 設定 `n_jobs=-1` 利用多核心
- 減少 CV folds（5→3）
- 使用較小的子集先練習

### Q2: RandomizedSearchCV 中 loguniform 無法 import
- scikit-learn >= 0.24 可直接使用 `sklearn.utils.fixes.loguniform`（已棄用）
- 建議使用 `scipy.stats.loguniform`：
```python
from scipy.stats import loguniform
```
- 若 scipy 版本較舊，可用：
```python
from scipy.stats import reciprocal  # loguniform 的舊名稱
```

### Q3: 學習曲線繪製時記憶體不足
- 減少 `train_sizes` 的點數
- 使用較小的資料集
- 確認 `n_jobs` 設定不會同時佔用過多記憶體

### Q4: 學生不理解學習曲線的含義
- 使用漸進式說明：
  1. 先用 10% 資料訓練，分數多少？
  2. 用 50% 資料訓練呢？
  3. 用 100% 呢？
  4. 把這些點連起來就是學習曲線
- 強調訓練分數和驗證分數的差距才是關鍵

### Q5: 學生對 Bayesian Optimization 的數學感到困惑
- 本週只需概念性理解，不需要推導
- 用比喻：「你是一個美食家，試過幾家餐廳後，你會根據經驗選擇下一家，而不是隨機挑」
- 將 Optuna 的使用作為加分題，不強制要求

### Q6: 超參數熱力圖看不出明顯的規律
- 可能是搜尋空間設計問題（範圍太窄或太寬）
- 引導學生調整搜尋範圍
- 這本身也是一個有價值的學習經驗
