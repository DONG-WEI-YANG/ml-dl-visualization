# Week 1 EnvironmentSetupViz 初學者友善增強設計

**日期**: 2026-03-09
**目標**: 讓 Week 1 視覺化更適合馬偕醫護專校初學者，疊加豐富內容並保持順暢

## 修改範圍

單一檔案：`platform/frontend/src/components/viz/EnvironmentSetupViz.tsx`

## 分頁結構（6 tabs）

### Tab 1：認識 AI（增強）
- **新增**：醫療情境引導故事（護理師量測數據 → 預測恢復狀況）
- **增強**：SVG 巢狀圓改為逐步展開動畫
  - 初始只顯示 AI 最外圈
  - 點擊展開 ML 中圈（CSS transition）
  - 再點擊展開 DL 內圈
  - 每層展開時自動顯示說明面板

### Tab 2：ML 工作流程（不變）

### Tab 3：動手預測（增強）
- **新增**：手動擬合模式
  - 兩個 slider 調整斜率 (w) 和截距 (b)
  - 即時顯示 MSE
  - 「顯示最佳解」按鈕 → 動畫過渡到最小平方法結果
  - 對比手動 MSE vs 最佳 MSE
- **保留**：原有點擊新增資料點功能

### Tab 4：過擬合體驗（全新）
- 固定散佈資料（~15 點，帶雜訊）+ 隱藏測試集
- slider 控制多項式次數（1 → 10）
- 即時繪製擬合曲線：欠擬合 → 剛好 → 過擬合
- 雙線圖顯示訓練誤差 vs 測試誤差
- 底部正則化概念說明

### Tab 5：名詞對照（增強）
- 預設只顯示「基礎概念」+「資料處理」（~16 詞）
- 進階名詞收在「顯示進階名詞」按鈕後

### Tab 6：環境建置（不變）

## 技術要點
- 多項式擬合用 Vandermonde matrix + 純前端計算
- 動畫用 CSS transition + React state
- 圖表用 Recharts（已引入）
- 全繁體中文
